它是怎么工作的？

监听提示音

检测指定应用在 Windows 音量混音器中的声音峰值， 用提示音作为提醒触发信号。

过滤前台应用

当你正在使用对应应用时，尽量避免因为语音、视频或当前窗口声音造成重复提醒。

弹出桌面横幅

在屏幕角落显示醒目的横幅提醒，点击后可尝试跳转到对应应用窗口。

为什么需要它？

有些 Windows 应用在收到消息时只会闪烁任务栏图标并播放提示音。 当你专注工作、没有佩戴耳机，或者正在使用多个窗口时，这些提醒很容易被忽略。 消息通知助手就是为了解决这个小但真实的问题。

隐私说明

消息通知助手不读取、不解析、不上传聊天内容。它不接入你的聊天账号， 也不破解或扫描 IM 软件的数据。它只检测你指定应用是否发出提示音， 并根据你的设置决定是否弹出提醒。

常见问题

为什么有时会误报？

本工具使用声音作为触发信号。如果某个应用的语音、视频或其他声音 与消息提示音难以区分，可能产生误报。

为什么不能显示消息内容？

为了降低侵入性和保护隐私，本工具不接入聊天账号， 也不读取 IM 软件的消息数据库或窗口内容。

为什么 Windows 会提示未知发布者？

当前安装包暂未进行代码签名，因此 Windows SmartScreen 可能提示未知发布者。 后续版本会考虑数字签名。

下载时 Edge 提示"通常不会下载"怎么办？

这是 Microsoft Edge / Windows SmartScreen 对新发布、下载量较低、暂未签名安装包的常见提示。 如果你是从本页面点击"下载最新版"获取安装包，可以在 Edge 下载面板中选择"保留"后继续。 请不要运行来自非官方页面或他人转发的不明安装包。

下载按钮没有反应怎么办？

可以尝试刷新页面后再次点击下载。若浏览器拦截了下载，请查看浏览器右上角的下载提示。
